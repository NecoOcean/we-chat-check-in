## 🎯 核心特性

- ✅ **权限隔离**: 市级管理员全部权限，县级管理员仅本县
- ✅ **县域验证**: 创建时验证县级管理员的县域匹配（v1.0.1）
- ✅ **名称唯一**: 同县域内教学点名称不能重复
- ✅ **软删除**: 删除仅标记状态，数据永久保留
- ✅ **县域名称**: 自动查询并返回县域名称（v1.0.1）
- ✅ **多条件查询**: 支持名称、县域、状态过滤
